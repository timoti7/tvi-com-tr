# tvi.com.tr

TVI Teknoloji kurumsal tanıtım sitesi.

## Tech Stack
- HTML5
- Tailwind CSS (CDN)
- Vanilla JS (minimal)

## Yapı
```
tvi.com.tr/
├── index.html      # Tek sayfa kurumsal site
├── assets/         # Görseller (logo vb.)
└── README.md
```

## Bölümler
1. **Hero** - Logo, tagline, CTA
2. **Hakkımızda** - Şirket hikayesi, rakamlar
3. **Ürünlerimiz** - 5 ürün kartı (BulunAI, Citation Studio, MüşteriMAI, Vizitmor, KolayPost)
4. **Kurucu** - Teoman Metehan profili
5. **İletişim** - Email, konum, form (frontend only)

## Local Development
Tarayıcıda `index.html` dosyasını açın.

## Deployment
Statik hosting için hazır (Railway, Netlify, Vercel, vb.)

## Notlar
- Light tema, kurumsal görünüm
- Türkçe içerik
- Form backend'i yok (sadece frontend)
